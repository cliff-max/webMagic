# webMagic ✨

> 一组有意思、有创意的 Web 交互效果合集。

## 简介

收集「vibe coding」风格的创意 Web 效果：3D 可视化、粒子手势交互、模拟水幕与水波纹、摄像头实感交互等。每个效果都是独立的小项目，互不依赖。

## 目录

```
webMagic/
├─ README.md
├─ LICENSE
├─ .gitignore
└─ demos/          # 所有 demo，各自独立
   └─ <demo-name>/
```

## 项目列表

- [particle-image](./demos/particle-image) — 图片转 3D 粒子云渲染模块：上传任意图片实时生成可交互 3D 粒子云，GPGPU 物理驱动、鼠标推动粒子、自由相机观测
